# 11.4 Zin 2 — een lijdend voorwerp

**Leerdoel:** je laat een werkwoord een **object** krijgen.

## De zin

> **Holmes (N) lit (V) a (Det) pipe (N).**

Nu doet Holmes iets *met iets*: hij steekt **een pijp** aan. Dat "a pipe"
is het **lijdend voorwerp** (het object van het werkwoord).

## Denk na

Je grammatica kan nu alleen nog een kale zin als zin 1. Twee dingen
ontbreken:

- Je werkwoord-groep moet een **object** achter het werkwoord kunnen
  hebben.
- "a pipe" is een **lidwoord (`Det`) + zelfstandig naamwoord (`N`)** —
  een nieuwe vorm van zelfstandig-naamwoordgroep.

Bedenk welke regels je daarvoor toevoegt. Zin 1 moet groen blijven!

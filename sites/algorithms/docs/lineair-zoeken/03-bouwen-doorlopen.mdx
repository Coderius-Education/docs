# 1.3 Bouwsteen 1 — door de lijst lopen

**Leerdoel:** je kunt **één voor één** door een lijst lopen in Python.
Dat is de eerste bouwsteen van lineair zoeken: het *één voor één*-stuk.

## Wat we willen

We willen elk element van de lijst kunnen bekijken. Niets zoeken nog,
gewoon: één voor één pakken en laten zien.

## Voorspel

Wat denk je dat dit print?

```python
lijst = [3, 1, 4, 1, 5]
for waarde in lijst:
    print(waarde)
```

<details>
<summary>Antwoord</summary>

```
3
1
4
1
5
```

De `for waarde in lijst:` betekent: *"haal achter elkaar elk element uit
de lijst, noem het `waarde`, en doe iets met dat element."*

</details>

## Wat nu nog mist

Dit *loopt door de lijst*, maar het *zoekt* nog niets. We weten nog niet
welk element we willen vinden — alleen dat we ze allemaal langskomen.

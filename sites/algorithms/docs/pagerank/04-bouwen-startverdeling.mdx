# 13.4 Bouwsteen 2 — de startverdeling

**Leerdoel:** je maakt de begin-PageRank: in het begin weet je nog niets,
dus elke pagina is even belangrijk.

## Eerlijk beginnen

Voordat je iets hebt uitgerekend, geef je elke pagina **dezelfde** rank:
`1 / N`, met `N` het aantal pagina's. Bij ons mini-web (4 pagina's) dus
`0.25` per pagina. Samen precies 1 — net als kansen.

Vanuit die startverdeling ga je straks itereren tot de waarden stabiel
zijn.

def startverdeling(web):
    n = len(web)
    return {p: 1 / n for p in web}

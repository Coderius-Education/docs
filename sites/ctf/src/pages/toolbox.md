# Toolbox

Een overzicht van handige tools die je nodig hebt bij de challenges. Allemaal gratis te gebruiken!

## Browser tools

### Developer Tools (DevTools)
**Ingebouwd in je browser** | Sneltoets: **F12** of **Ctrl+Shift+I**

De belangrijkste tool voor webgerelateerde challenges. Hiermee kun je:
- **Elements**: HTML en CSS van een pagina inspecteren en aanpassen
- **Console**: JavaScript uitvoeren en foutmeldingen bekijken
- **Network**: HTTP-verzoeken en -antwoorden bekijken, inclusief headers
- **Application**: Cookies bekijken en wijzigen
- **Sources**: Bronbestanden van de pagina bekijken

### Paginabron bekijken
**Sneltoets: Ctrl+U**

Toont de volledige HTML-broncode van een pagina. Handig om verborgen elementen, commentaren of scripts te vinden.

## Online tools

### CyberChef
[gchq.github.io/CyberChef](https://gchq.github.io/CyberChef/)

Het "Zwitserse zakmes" voor data-analyse. Kan vrijwel alles:
- Base64 encoderen/decoderen
- ROT13 en andere Caesar ciphers
- Hex naar tekst converteren
- Automatische detectie met de "Magic" functie
- En nog honderden andere operaties

### VirusTotal
[virustotal.com](https://www.virustotal.com/)

Scan bestanden en URLs met tientallen antivirusprogramma's tegelijk. Bekijk ook de details, metadata en community-commentaren.

### Base64 Decode
[base64decode.org](https://www.base64decode.org/)

Simpele tool specifiek voor Base64 decodering. Plak de gecodeerde tekst en krijg direct het resultaat.

### ROT13 Decoder
[rot13.com](https://rot13.com/)

Specifiek voor ROT13 decodering. Werkt ook voor andere rotaties (ROT1 t/m ROT25).

### IP Geolocatie
[iplocation.net](https://www.iplocation.net/)

Zoek de geografische locatie op van een IP-adres. Toont stad, regio en land.

### Wayback Machine
[web.archive.org](https://web.archive.org)

Bekijk gearchiveerde versies van websites. Voer een URL in en blader door snapshots uit het verleden.

## Software (downloaden)

### Audacity
[audacityteam.org](https://www.audacityteam.org/) | **Gratis, open-source**

Audio-editor waarmee je geluidsfragmenten kunt:
- Omkeren (reverse)
- Versnellen of vertragen
- Spectrogram bekijken
- Effecten toepassen

### Hex Editor
Bijvoorbeeld [HxD](https://mh-nexus.de/en/hxd/) (Windows) | **Gratis**

Bekijk de ruwe bytes van een bestand. Handig om bestandstypen te identificeren aan de hand van magic bytes.

## Browser Console cheatsheet

Handige JavaScript-commando's die je kunt uitvoeren in de browserconsole (F12 > Console):

```javascript
// Base64 decoderen
atob("SGVsbG8gV29ybGQ=")

// Base64 encoderen
btoa("Hello World")

// Alle cookies bekijken
document.cookie
```

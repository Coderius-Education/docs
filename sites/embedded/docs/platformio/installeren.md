# 7.2 PlatformIO installeren

PlatformIO draait als uitbreiding in de editor **VS Code**. Je installeert dus eerst VS Code, daarna de uitbreiding.

## Stap 1: VS Code installeren

Download VS Code van [code.visualstudio.com](https://code.visualstudio.com) en installeer het.

## Stap 2: De PlatformIO-uitbreiding installeren

1. Open VS Code.
2. Klik in de linkerbalk op het **Extensions**-icoon (de vier blokjes).
3. Zoek op **PlatformIO IDE**.
4. Klik op **Install**.

De eerste installatie duurt even: PlatformIO downloadt op de achtergrond de gereedschappen die het nodig heeft. Wacht tot de melding verschijnt dat het klaar is, en herstart VS Code als daarom wordt gevraagd.

## Stap 3: Het PlatformIO-startscherm

Na het herstarten zie je in de linkerbalk een nieuw icoon: het **mierenhoofd** van PlatformIO. Klik erop en kies **PIO Home → Open**. Dit startscherm gebruik je om nieuwe projecten te maken.

:::tip
Heb je een echt bord? Sluit het nu alvast aan via USB. Op Windows installeert PlatformIO meestal zelf de juiste driver. Lukt uploaden later niet, kijk dan bij [Upload mislukt](/docs/er-gaat-iets-mis/upload-mislukt).
:::

In de volgende les maak je je eerste PlatformIO-project: opnieuw een blink, maar nu in deze nieuwe omgeving.

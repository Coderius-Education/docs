# 0 Welkom bij Python

Wat leuk dat je begint met Python! Python is een van de populairste programmeertalen ter wereld. Je vindt het terug in websites, games, data-analyse en kunstmatige intelligentie. En het mooiste: het is een prima taal om mee te beginnen.

## Wat ga je leren?

Je werkt **één concept per les** door, zodat je nooit wordt overspoeld. Onderwerpen die je tegenkomt:

- **Print en variabelen** — je eerste programma, en informatie opslaan
- **Rekenen** — operatoren en rekenvolgorde
- **Slimme berichten** — f-strings en string-methoden
- **Beslissen** — booleans, `if`/`else`, `and`/`or`, `elif`
- **Herhalen** — `for`-loops en `while`-loops
- **Functies** — `def`, parameters en `return`
- **Lijsten en dictionaries** — meerdere waardes bijhouden

## Hoe werkt elke les?

Elke les heeft dezelfde opbouw:

1. Een korte intro die aansluit op wat je al kent.
2. Een minimaal werkend codevoorbeeld.
3. Een paar zinnen uitleg over wat er gebeurt.
4. Twee tot vier opdrachten om het zelf te doen, met een tip en de oplossing in een uitklap-blok.

Je hoeft niets te installeren — alle code draai je direct in de browser.

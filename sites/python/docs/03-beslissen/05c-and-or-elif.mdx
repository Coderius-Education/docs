# 5c And, or en elif

Je kunt nu kiezen tussen twee paden met `if`/`else`. In deze les voeg je twee dingen toe: meerdere voorwaarden combineren met `and` en `or`, en meer dan twee takken met `elif`.

## Twee voorwaarden combineren: and en or

| Operator | Wanneer `True`? |
|:--------:|-----------------|
| `and`    | Als **beide** voorwaarden `True` zijn |
| `or`     | Als **minstens Ă©Ă©n** voorwaarde `True` is |

```python
leeftijd = 16
heeft_kaartje = True

if leeftijd >= 14 and heeft_kaartje:
    print("Je mag naar binnen!")
else:
    print("Sorry, je mag er niet in.")
```

Vraag jezelf bij twijfel: "moet allebei, of mag Ă©Ă©n?" â€” dat bepaalt of je `and` of `or` kiest.

## Meer dan twee takken: elif

`if`/`else` geeft je twee opties. Met `elif` (een afkorting van *else if*) voeg je extra takken toe:

```python
score = 65

if score >= 80:
    print("Uitstekend!")
elif score >= 55:
    print("Voldoende.")
else:
    print("Helaas, onvoldoende.")
```

Python kijkt de voorwaarden **van boven naar beneden** na en stopt zodra er Ă©Ă©n `True` is. De andere takken worden dan niet meer gecheckt. Je kunt zoveel `elif`-takken plaatsen als je wilt.

## Opdracht 5c.3: Strenge toegangscontrole

Bouw een toegangscontrole die per reden uitlegt waarom iemand niet binnen mag. Gebruik `op_lijst` en `leeftijd` (drempel: 14).

Verwachte uitvoer (bij `op_lijst = False`, `leeftijd = 12`):

```
Sorry Alex, je staat niet op de lijst Ă©n je bent te jong.
```

Verwachte uitvoer (bij `op_lijst = False`, `leeftijd = 18`):

```
Sorry Alex, je staat niet op de gastenlijst.
```

Verwachte uitvoer (bij `op_lijst = True`, `leeftijd = 12`):

```
Sorry Alex, je bent te jong (minimum: 14).
```

Verwachte uitvoer (bij `op_lijst = True`, `leeftijd = 18`):

```
Welkom Alex!
```

<details>
<summary>Klik hier voor een tip!</summary>

`not op_lijst` is `True` als de gast **niet** op de lijst staat. Vier takken op volgorde:

1. `not op_lijst and leeftijd < 14` â†’ beide problemen
2. `not op_lijst` â†’ alleen lijst-probleem
3. `leeftijd < 14` â†’ alleen leeftijd-probleem
4. `else` â†’ welkom

</details>

<details>
<summary>Klik hier voor de oplossing!</summary>

```python
naam = "Alex"
leeftijd = 12
op_lijst = False

if not op_lijst and leeftijd < 14:
    print(f"Sorry {naam}, je staat niet op de lijst Ă©n je bent te jong.")
elif not op_lijst:
    print(f"Sorry {naam}, je staat niet op de gastenlijst.")
elif leeftijd < 14:
    print(f"Sorry {naam}, je bent te jong (minimum: 14).")
else:
    print(f"Welkom {naam}!")
```

In de volgende les leer je de **for-loop**: een rijtje getallen of namen aflopen zonder elke regel zelf te typen.

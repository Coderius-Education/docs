# Stap 3: een Python-bestand aanmaken

1. Klik met de rechtermuisknop in de Explorer (het linker paneel)
2. Kies **New File...**
3. Noem het bestand `main.py`
4. Typ de volgende code in het bestand:

```python
print("Hallo wereld!")
```

5. Sla het bestand op via **File** → **Save**

# Stap 1: Python installeren

Voor Python in VS Code heb je twee dingen nodig: **Python zelf** op je computer, en de **Python-extensie** in VS Code.

## Python op je computer

1. Ga naar [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. Download de versie van Python die je nodig hebt. Check heel goed bij je project welke versie nodig is.
3. **Belangrijk:** vink tijdens de installatie het vakje **"Add Python to PATH"** aan
4. Klik op **Install Now**

## Python-extensie in VS Code

Zonder de Python-extensie van Microsoft werkt de **▶ Play-knop** niet en mist VS Code het commando om een virtual environment aan te maken.

1. Open VS Code
2. Klik in de linkerbalk op het **blokjes-icoon** (Extensions)
3. Typ bovenaan in het zoekveld: `Python`
4. Zoek het resultaat van **Microsoft** (de eerste in de lijst, met een blauw vinkje)
5. Klik op de blauwe **Install** knop

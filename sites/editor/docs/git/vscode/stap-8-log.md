# Stap 8: controleer met `git log`

Open de terminal in VS Code en typ:

```bash
git log
```

Je ziet beide commits, nieuwste eerst. Dit is exact hetzelfde commando als in de simulator — alleen draait het nu op je echte computer met echte bestanden.

Druk op **q** om het log te sluiten.

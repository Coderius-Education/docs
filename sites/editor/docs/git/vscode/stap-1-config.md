# Stap 1: vertel git wie je bent

Eenmalig moet je git vertellen wie commits maakt. Open in VS Code een terminal via **Terminal** → **New Terminal** en typ:

```bash
git config --global user.name "Jouw Naam"
git config --global user.email "jouw@email.nl"
```

Vervang de naam en het e-mailadres door je eigen gegevens. Dit slaat git op voor al je toekomstige projecten.

# Stap 7: een tweede commit

Pas `hello.txt` aan: voeg een tweede regel toe, bijvoorbeeld `Dit is regel twee.` en sla op.

Kijk weer naar **Source Control**. Naast `hello.txt` staat nu een **M** — **Modified** (gewijzigd). Stage hem (`+`), typ een nieuwe commitboodschap (`tweede regel toegevoegd`) en commit.

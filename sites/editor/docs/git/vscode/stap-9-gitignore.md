# Stap 9: een `.gitignore` aanmaken

Maak een nieuw bestand in je map met de naam `.gitignore`. Zet erin (één regel):

```
geheim.txt
```

Maak nu een bestand `geheim.txt` aan met wat tekst erin. Kijk naar **Source Control**: `geheim.txt` verschijnt **niet** in de lijst. Git negeert het.

# Stap 2: een projectmap openen

1. Maak een nieuwe map aan op je computer, bijvoorbeeld `git-oefenen`
2. Maak in die map een leeg bestand `hello.txt` met de tekst `Hallo wereld!`
3. Open de map in VS Code via **File** → **Open Folder...**

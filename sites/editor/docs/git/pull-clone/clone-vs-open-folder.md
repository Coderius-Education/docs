# Clone vs. Open Folder

Twee knoppen die op elkaar lijken maar iets heel anders doen:

| Knop | Wat het doet |
|:---|:---|
| **File** → **Open Folder...** | Opent een map die al op je computer staat. Doet niets met git. |
| **Command Palette** → **Git: Clone** | Downloadt een complete repository van GitHub (of een andere git-server) inclusief alle commits, en opent die. |

Als je **Open Folder** doet op een map die je eerder gekloond hebt, werkt git ook gewoon — git was er al, je opent hem alleen opnieuw.

# Welke knop is welk commando?

| In VS Code | Doet hetzelfde als |
|:---|:---|
| **Sync Changes** | `git pull` + `git push` |
| **Command Palette** → **Git: Pull** | `git pull` |
| **Command Palette** → **Git: Clone** | `git clone <url>` |

## Wat je tot nu toe kunt

Je beheerst nu de pull/push-cyclus: lokaal committen, online zetten, weer ophalen, en bestaande projecten clonen. Daarmee kun je je eigen werk veilig opslaan en op meerdere computers werken.

Voor samenwerken — of voor netjes werken op je eigen projecten — kom je nog twee dingen tegen: **branches** (parallel werken zonder `main` te breken) en **pull requests** (een branch via GitHub mergen).

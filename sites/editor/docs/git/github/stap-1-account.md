# Stap 1: een GitHub-account aanmaken

1. Ga naar [https://github.com](https://github.com)
2. Klik rechtsboven op **Sign up**
3. Kies een gebruikersnaam, e-mailadres en wachtwoord
4. Bevestig je e-mailadres via de link die GitHub stuurt

**Tip:** kies een gebruikersnaam die je later nog leuk vindt. Veel programmeurs gebruiken hun GitHub-naam jarenlang.

**Let op:** vul bij je gebruikersnaam geen schoolgegevens of je volledige geboortedatum in. Je profiel is openbaar.

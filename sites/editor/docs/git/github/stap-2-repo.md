# Stap 2: een lege repository aanmaken

1. Log in op github.com
2. Klik rechtsboven op het **`+`**-icoon → **New repository**
3. Vul in:
   - **Repository name:** bijvoorbeeld `git-oefenen`
   - **Description:** kort wat erin staat (optioneel)
   - **Public** of **Private** — kies wat je wilt. **Public** is zichtbaar voor iedereen, **Private** alleen voor jou.
4. **Belangrijk:** vink **niks** aan bij "Initialize this repository with..." (geen README, geen `.gitignore`, geen license).

   Anders krijg je bij je eerste push een conflict — GitHub heeft dan al een commit en jij ook, en die kan git niet zomaar combineren.
5. Klik op **Create repository**

Je ziet nu een lege repository-pagina met instructies. Laat dit venster open — je hebt de URL straks nodig.

# Stap 3: rondkijken op je repo-pagina

Klik door deze knoppen op je nieuwe repository-pagina:

- **Code** (groene knop) — hier vind je straks de URL die je nodig hebt om te koppelen of te clonen.
- **Settings** (rechtsboven) — instellingen van de repo, bijvoorbeeld de naam veranderen of de repo verwijderen.
- **commits** (kloktje-icoon, ergens bovenaan) — staat nu op `0 commits`. Zodra je iets pusht, verschijnen je commits hier.
- **README**-instructies — GitHub toont een aantal terminal-commando's die je *zou* kunnen typen. Geen zorgen, je gaat dit straks via VS Code doen.

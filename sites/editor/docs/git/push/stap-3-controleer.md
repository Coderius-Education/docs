# Stap 3: controleer op GitHub

1. Open je repository-pagina op github.com
2. Klik bovenaan op het kloktje-icoon (**commits**)
3. Je ziet alle commits met hun boodschap, datum en auteur — net als `git log`
4. Klik op een commit-boodschap om te zien welke regels precies veranderd zijn

Dit is een van de prettigste dingen aan GitHub: je geschiedenis is overal en altijd te bekijken, ook op je telefoon.

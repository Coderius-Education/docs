# Welke knop is welk commando?

| In VS Code | Doet hetzelfde als |
|:---|:---|
| Statusbalk-knop → **+ Create new branch from...** | `git checkout -b <naam>` |
| Statusbalk-knop → andere branch kiezen | `git checkout <naam>` |
| Source Control **...** → **Branch** → **Merge Branch...** | `git merge <branch>` |
| **Command Palette** → **Git: Delete Branch...** | `git branch -d <branch>` |

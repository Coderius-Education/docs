# Git basis

In deze tutorial leer je de basis van **git**: een programma waarmee je versies van je code bijhoudt. Je oefent eerst in een veilige simulator in je browser. Daarna leer je in [Git in VS Code](/git/vscode/) hetzelfde te doen op je eigen computer.

## Wat is git?

**Git** is een **versiebeheer**-systeem. Stel: je hebt een werkende website en je gaat hem aanpassen. Wat als je nieuwe versie kapot gaat? Met git kun je elke werkende versie als "snapshot" opslaan. Een snapshot heet een **commit**. Je kunt altijd terug naar een vorige commit.

Git werkt met drie zones:

- **Werkmap** — de bestanden waar je op dit moment in zit te typen.
- **Staging** — een soort wachtkamer voor wijzigingen die je in de volgende commit wilt meenemen.
- **Repository** — alle opgeslagen commits (de geschiedenis).

In elke stap van deze tutorial zie je een simulator. Links zie je de drie zones. Rechts is een terminal waarin je `git`-commando's typt.

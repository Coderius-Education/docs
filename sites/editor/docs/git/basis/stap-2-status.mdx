# Stap 2: kijken wat git ziet met `git status`

`git status` is je meest gebruikte commando. Het vertelt je welke bestanden veranderd zijn, welke al klaarliggen voor een commit, en welke git nog niet kent.

Een nieuw bestand dat git nog niet kent heet **untracked** (niet-gevolgd).

```bash
git status
```

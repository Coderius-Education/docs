# Installatie VS Code

In deze tutorial leer je hoe je Visual Studio Code (VS Code) installeert en klaar maakt voor gebruik.

## Wat is VS Code?

Visual Studio Code is een gratis code-editor van Microsoft. Het is lichtgewicht, snel en heeft heel veel handige functies zoals syntax highlighting, autocompletion en een ingebouwde terminal.

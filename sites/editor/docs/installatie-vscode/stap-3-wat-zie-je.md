# Stap 3: wat zie je nu?

Wanneer je VS Code voor het eerst opent, zie je het volgende:

- **Explorer** (links) — hier zie je de bestanden en mappen van je project
- **Editor** (midden) — hier bewerk je je bestanden
- **Terminal** (onder) — een ingebouwde terminal, te openen via **Terminal** → **New Terminal**
- **Statusbalk** (helemaal onder) — informatie over je bestand en project

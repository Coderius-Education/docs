# Stap 1: download VS Code

1. Ga naar de officiële website: [https://code.visualstudio.com](https://code.visualstudio.com)
2. Klik op de grote blauwe **Download** knop
3. De website detecteert automatisch je besturingssysteem (Windows, macOS of Linux)

# Stap 5: een bestand aanmaken

1. Klik met de rechtermuisknop in de Explorer (het linker paneel)
2. Kies **New File...**
3. Geef het bestand een naam, bijvoorbeeld `test.txt`
4. Sla het bestand op via **File** → **Save**

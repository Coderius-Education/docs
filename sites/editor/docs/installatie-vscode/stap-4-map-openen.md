# Stap 4: een map openen

Om te beginnen met coderen heb je een projectmap nodig:

1. Klik op **File** → **Open Folder...**
2. Kies een map op je computer (of maak een nieuwe aan)
3. Klik op **Map selecteren**

Je ziet nu de bestanden in het linker paneel onder **Explorer**.

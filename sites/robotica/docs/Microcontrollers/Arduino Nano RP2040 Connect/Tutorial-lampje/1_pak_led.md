# 3.1 Materiaal

Voor deze tutorial heb je nodig:

1. Arduino Nano RP2040 Connect
2. Een **LED-lampje** (extern, met twee pootjes)

![led](led.png)

Let goed op: een LED heeft een **lange** en een **korte** pin. De lange pin is de **plus** (+), de korte pin is de **min** (-).

<details>
<summary>Controlevraag</summary>

Welke pin van de LED gaat naar GND?

</details>

<details>
<summary>Antwoord</summary>

De **korte** pin (de min, -). De **lange** pin gaat naar een digitale uitgang van de microcontroller.

</details>

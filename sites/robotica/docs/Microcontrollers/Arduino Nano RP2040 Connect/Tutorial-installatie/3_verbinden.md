# 1.3 Bord verbinden met je computer

Voordat je code kunt sturen, moet de microcontroller met je computer praten via een USB-kabel.

## Wat heb je nodig?

1. Een USB-kabel van **USB-A naar micro-USB**
2. De Arduino Nano RP2040 Connect

![frame](@site/static/fritzing/nano_rp2040_verbinden_bb.png)

## Aansluiten

Steek de **USB-A**-kant in je computer en de **micro-USB**-kant in de microcontroller. Als alles werkt, gaat er een **groen lampje** branden op het bord.

<details>
<summary>Tip: ik zie geen groen lampje</summary>

Probeer een andere USB-poort of een andere kabel. Sommige kabels zijn alleen geschikt om op te laden en kunnen geen data versturen.

</details>

<details>
<summary>Controlevraag</summary>

Hoe weet je dat de verbinding goed is?

</details>

<details>
<summary>Antwoord</summary>

Het **groene lampje** op de Arduino Nano RP2040 Connect brandt zodra het bord stroom krijgt via USB.

</details>

# 1.2 Een editor installeren

Om Python-code te schrijven heb je een editor nodig: een programma waarin je je code typt en runt.

:::tip Geen installatie nodig

Werk je in Chrome of Microsoft Edge? Dan kun je ook de [online editor](/editor) gebruiken: typen, op **Run op board** klikken, klaar. Geen installatie nodig.

Werkt dat niet (andere browser, schoollaptop met restricties)? Volg dan de Thonny-stappen hieronder.

:::

Op deze pagina installeren we **Thonny** als fallback en lokaal alternatief. Die werkt meteen met MicroPython op je microcontroller.

## Stap 1: download Thonny

Ga naar [thonny.org](https://thonny.org/) en kies je besturingssysteem (meestal Windows). Download het bestand dat begint met **"Installer with 64-bit Python..."**.

## Stap 2: installeren

Voer het installatiebestand uit en klik de installatie door. Open daarna Thonny.

<details>
<summary>Controlevraag</summary>

Wat zie je als Thonny voor het eerst opent?

</details>

<details>
<summary>Antwoord</summary>

Een leeg codescherm bovenin en onderaan de **Shell** (ook wel REPL genoemd). In die Shell verschijnt straks de uitvoer van je code.

</details>

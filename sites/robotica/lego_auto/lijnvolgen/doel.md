# Wat is de bedoeling?

In dit hoofdstuk laat je je robot een **zwarte lijn volgen**. Alles wat je tot nu toe geleerd hebt (IR-sensoren, motoren, while-loops) komt hier samen.

## Wanneer heb je deze stap gehaald?

- Je robot volgt een zwarte lijn op een witte ondergrond.

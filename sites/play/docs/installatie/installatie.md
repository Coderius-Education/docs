# Installatie

Om met coderius-play te werken, heb je drie dingen nodig:

1. **Een editor** — een programma waarin je code typt en uitvoert
2. **Python** — de programmeertaal
3. **coderius-play** — de bibliotheek waarmee je games maakt

Hieronder geven we drie opties. Kies er één of vraag je docent om advies.

## Optie 1: Online speeltuin (geen installatie nodig)

Wil je meteen beginnen zonder iets te installeren? Gebruik dan de [online speeltuin](/speeltuin). Je typt je code direct op de pagina en klikt op de afspeelknop — geen downloads, geen gedoe.

:::info Let op
De online speeltuin werkt alleen binnen dit lesmateriaal. Voor een eigen project dat je lokaal opslaat en uitvoert, heb je toch een editor en Python nodig (zie de opties hieronder).
:::

## Optie 2: Thonny

Wil je gewoon snel aan de slag? Kies dan voor Thonny. Het is een simpele editor die speciaal gemaakt is voor beginners. Python zit er al bij inbegrepen!

Ga naar [de Thonny-tutorial](Thonny.md) voor de stappen.

## Optie 3: Visual Studio Code

Wil je een editor die er mooier uitziet en ook veel meer kan? Kies dan voor VS Code. De installatie is iets lastiger, maar het resultaat is een krachtigere werkomgeving.

Ga naar [de VS Code-tutorial](VScode.md) voor de stappen.

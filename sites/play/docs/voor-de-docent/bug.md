# Foutje gevonden?

Goed speurwerk! Er zijn twee soorten foutjes die je kunt tegenkomen:

## 1. Een bug in de software

De software doet iets wat niet klopt. Bijvoorbeeld: een vorm beweegt niet zoals verwacht, of je krijgt een foutmelding terwijl je code correct is.

Meld dit op de **play** repository:
[Issues voor software bugs](https://github.com/Coderius-Education/play/issues)

## 2. Een foutje in het lesmateriaal

Er staat iets dat niet klopt in de uitleg of opdrachten. Bijvoorbeeld: een typfout, een verkeerd codevoorbeeld, of een onduidelijke uitleg.

Meld dit op de **play-docs** repository:
[Issues voor het lesmateriaal](https://github.com/Coderius-Education/play-docs/issues)

## 3. Hoe maak je een Issue aan?

Je gaat een zogenaamd `GitHub Issue` aanmaken.
Je kunt onderstaande video gebruiken om te leren hoe je een issue kunt aanmaken:

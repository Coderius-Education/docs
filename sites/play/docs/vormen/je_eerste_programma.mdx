# 1.1 Je eerste programma schrijven en uitvoeren

Het is tijd voor je eerste programma!

## Stap 1: Open je editor
- **Online speeltuin**: geen installatie nodig, ga naar [de speeltuin](/speeltuin).
- **Thonny**: open Thonny op je computer.
- **VS Code**: open je VS Code editor op je computer

## Stap 2: Schrijf je eerste programma
- **Online speeltuin**: de code staat al klaar hieronder, klik direct op ▶.
- **Thonny / VS Code**: maak een nieuw bestand aan, sla het op als `main.py` en typ de volgende code:

<PygbagRunner code={`import play

play.new_circle()`} height={300} />

## Stap 3: Voer het programma uit
- **Online speeltuin**: klik op de ▶ knop in de speeltuin.
- **Thonny**: klik op de groene knop (`Run current script`) om de code te draaien.
- **VS Code**: klik rechtsboven op de ▶ knop, of gebruik de terminal om het bestand uit te voeren.

Als het goed is, zie je een zwarte cirkel op een witte achtergrond. Bij de online speeltuin gebeurt dit in de browser; bij VS Code of Thonny opent er een venster op je computer (standaard 800 bij 600 pixels).
Je mag op het kruisje klikken om dit af te sluiten.
